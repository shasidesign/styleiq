╔══════════════════════════════════════════════════════════════════════════╗
║           STEP 1 — SET UP MONGODB ATLAS (FREE FOREVER)                 ║
╚══════════════════════════════════════════════════════════════════════════╝

MongoDB Atlas gives you a FREE 512MB cloud database forever.
No credit card needed for the free tier (M0).

─── 1.1 Create Account ────────────────────────────────────────────────────

1. Open browser → go to:  https://www.mongodb.com/cloud/atlas/register

2. Fill in:
   • Email address (use your Gmail or college email)
   • First Name, Last Name
   • Password
   • Click "Create your Atlas account"

3. Verify your email (check inbox, click the link)

4. Answer the onboarding questions:
   • "What is your goal?" → Choose: "Learn MongoDB"
   • "What type of application?" → Choose: "Web Application"
   • Click "Finish"

─── 1.2 Create Free Cluster ───────────────────────────────────────────────

1. You'll see "Deploy your database" screen
2. Choose "M0 FREE" (it says FREE, 512 MB)
3. Provider: choose AWS (default is fine)
4. Region: choose the one CLOSEST to your location
   • India → Mumbai (ap-south-1)
   • US → N. Virginia (us-east-1)
   • Europe → Frankfurt (eu-central-1)
5. Cluster Name: type "StyleIQ" (or keep default "Cluster0")
6. Click "Create Deployment" (big green button)
7. WAIT ~2 minutes for cluster to deploy ⏳

─── 1.3 Create Database User ──────────────────────────────────────────────

A popup "Connect to Cluster0" will appear automatically.

1. Username: type "styleiq_user"
2. Password: click "Autogenerate Secure Password"
   → COPY THIS PASSWORD and save it somewhere safe!
   Or use your own: e.g. "StyleIQ2024@secure"
3. Click "Create Database User"
4. Click "Choose a connection method" (or "Next")

─── 1.4 Add IP Whitelist ──────────────────────────────────────────────────

1. In the popup, click "Add my current IP Address"
   (This allows your laptop to connect)
2. Also add: 0.0.0.0/0 — click "Add Entry" → type 0.0.0.0/0 → Add
   (This allows Vercel/any server to connect — needed for deployment)
3. Click "Finish and Close"

─── 1.5 Get Your Connection String ────────────────────────────────────────

1. Back on Atlas dashboard, click "Connect" button on your cluster
2. Choose "Drivers"
3. Driver: Python | Version: 3.12 or later
4. Copy the connection string. It looks like:

   mongodb+srv://styleiq_user:<password>@cluster0.abc123.mongodb.net/?retryWrites=true&w=majority

5. Replace <password> with your actual password
6. Add the database name before the "?" like this:

   mongodb+srv://styleiq_user:YourPassword@cluster0.abc123.mongodb.net/styleiq?retryWrites=true&w=majority

SAVE THIS — you'll use it in Step 3.

─── 1.6 Create Collections (Optional - auto-created) ─────────────────────

StyleIQ creates collections automatically. But you can create them manually:

1. Click "Browse Collections" on your cluster
2. Click "Add My Own Data"
3. Database name: "styleiq"
4. Create these collections one by one:
   • users
   • predictions
   • favorites
   • fashion_products
   • chatbot_history
   • analytics

╔══════════════════════════════════════════════════════════════════════════╗
║           STEP 3 — LOCAL SETUP & RUN                                   ║
╚══════════════════════════════════════════════════════════════════════════╝

─── 3.1 Install Python ────────────────────────────────────────────────────

Download Python 3.11+ from: https://www.python.org/downloads/
• Windows: download .exe installer, check "Add to PATH" ✓
• Mac: download .pkg installer
• Verify: open Terminal/CMD, type: python --version

─── 3.2 Extract & Setup Project ───────────────────────────────────────────

# Open Terminal (Mac/Linux) or Command Prompt (Windows)

# 1. Extract the zip (or navigate to extracted folder)
cd styleiq

# 2. Create virtual environment (keeps dependencies isolated)
python -m venv venv

# 3. Activate virtual environment
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# You should see (venv) at the start of your terminal line

─── 3.3 Install Dependencies ──────────────────────────────────────────────

pip install -r requirements.txt

# This installs: Flask, PyMongo, scikit-learn, numpy, werkzeug, python-dotenv, openai, gunicorn
# Takes about 2-3 minutes ⏳

─── 3.4 Configure Environment ─────────────────────────────────────────────

# Copy the example .env file
# Windows:
copy .env.example .env
# Mac/Linux:
cp .env.example .env

# Now EDIT the .env file with your actual values:
# Open with Notepad (Windows) or nano/code (Mac/Linux)

# Edit these lines:
SECRET_KEY=any-long-random-string-like-styleiq2024secretibmproject
MONGO_URI=mongodb+srv://styleiq_user:YourPassword@cluster0.abc123.mongodb.net/styleiq?retryWrites=true&w=majority
OPENAI_API_KEY=sk-your-actual-key-here

─── 3.5 Run the App ───────────────────────────────────────────────────────

python app.py

# You should see:
# ✅ MongoDB connected successfully
# * Running on http://127.0.0.1:5000
# * Debug mode: on

# Open browser: http://localhost:5000
# 🎉 StyleIQ is running!

─── 3.6 Seed Demo Products ────────────────────────────────────────────────

Run this once to add products to MongoDB:
curl -X POST http://localhost:5000/api/seed-products

Or visit: http://localhost:5000/api/seed-products (POST via Postman/browser extension)


╔══════════════════════════════════════════════════════════════════════════╗
║           STEP 4 — DEPLOY TO VERCEL (FREE HOSTING)                     ║
╚══════════════════════════════════════════════════════════════════════════╝

Vercel gives you a live URL like: https://styleiq.vercel.app

─── 4.1 Create GitHub Account ─────────────────────────────────────────────

1. Go to: https://github.com
2. Sign up with your email
3. Verify email

─── 4.2 Upload Project to GitHub ──────────────────────────────────────────

# Install Git from: https://git-scm.com/downloads

# In your project folder (Terminal):
git init
git add .
git commit -m "StyleIQ - IBM AI Fashion Platform"

# Go to github.com → New Repository
# Name: styleiq
# Private: YES (to protect your .env!)
# Create repository

# Copy the commands GitHub shows, they look like:
git remote add origin https://github.com/yourusername/styleiq.git
git branch -M main
git push -u origin main

IMPORTANT: Make sure .env is in .gitignore (it already is in this project)
NEVER push your .env file to GitHub!

─── 4.3 Deploy on Vercel ──────────────────────────────────────────────────

1. Go to: https://vercel.com
2. Sign up with your GitHub account
3. Click "Add New Project"
4. Click "Import" next to your styleiq repository
5. Framework Preset: "Other" (Vercel auto-detects Python)
6. Click "Deploy" → wait ~2 minutes

─── 4.4 Set Environment Variables on Vercel ───────────────────────────────

After deploy (even if it fails first time):

1. Go to your project on Vercel dashboard
2. Click "Settings" tab
3. Click "Environment Variables" (left sidebar)
4. Add these one by one:

   Name: SECRET_KEY
   Value: your-super-secret-key-here

   Name: MONGO_URI
   Value: mongodb+srv://styleiq_user:YourPassword@cluster0.abc123.mongodb.net/styleiq?retryWrites=true&w=majority

   Name: OPENAI_API_KEY
   Value: sk-your-key-here

5. Click "Save" after each
6. Go to "Deployments" tab → click "..." → "Redeploy"
7. Your app is LIVE at: https://styleiq-yourusername.vercel.app

─── 4.5 Verify Live Deployment ────────────────────────────────────────────

Visit your Vercel URL and test:
✓ Homepage loads
✓ Register/Login works
✓ AI Predictor returns results
✓ Products load (may need to seed: POST /api/seed-products)
✓ StyleBot responds
✓ Dashboard shows data

╔══════════════════════════════════════════════════════════════════════════╗
║           TROUBLESHOOTING                                               ║
╚══════════════════════════════════════════════════════════════════════════╝

Problem: "ModuleNotFoundError"
Fix: pip install -r requirements.txt  (make sure venv is activated)

Problem: "MongoDB connection failed"
Fix: Check MONGO_URI in .env — password must not have special characters
     Encode special chars: @ → %40, # → %23

Problem: "OpenAI API error"
Fix: The app automatically uses rule-based chatbot. No action needed.
     Or update key in .env

Problem: App works locally but not on Vercel
Fix: Check all 3 environment variables are set in Vercel dashboard
     Check Vercel function logs for error details

Problem: Products not showing
Fix: POST to /api/seed-products to load demo products into MongoDB

Problem: "pip not found"  
Fix: Use "pip3" instead of "pip"
     Or: python -m pip install -r requirements.txt
